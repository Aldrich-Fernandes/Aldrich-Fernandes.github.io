Mortgage Egibility Checker

A mortgage eligibility checker, built entirely from scratch in Python with minimal imports for my A-Level NEA project. The tool uses a logistic regression model embedded within a neural network framework to evaluate the strength of mortgage applications based on historical applicant data. Functionality includes model saving/loading, hyperparameter tuning, and efficient data preprocessing, making it both practical and adaptable for real-world decision-making.

Key Features

  • User-Friendly Interface - Simple, intuitive input system for financial and personal details.
  • Real-Time Predictions - Instant feedback on loan approval likelihood using custom-built ML models.
  • Customizable Model Training - Adjustable hyperparameters and training settings for experimentation.
  • Model Persistence - Save and load trained models for repeated use without retraining.
  • Data Preprocessing Pipeline - Streamlined handling of applicant data to ensure clean, consistent inputs.
  • From-Scratch Neural Network - Logistic regression within a neural framework coded without external ML libraries.
